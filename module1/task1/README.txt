Completion of task 1 of module 1:
1. Implement function which accepts a string and returns its hash asynchronously. This function should be called for each element of the array.
To view this feature you need to enter command in the terminal: " npm run asynchronously "
2. Implement npm script to process the array sequentially using “for” statement.
To view this feature you need to enter command in the terminal: " npm run sequentially "
3. Implement npm script to process the array sequentially using function calls only (no loops).
To view this feature you need to enter command in the terminal: " npm run calls "
4. Implement npm script to process the array in batches sequentially. Choose the batch size on your own.
To view this feature you need to enter command in the terminal: " npm run batches "
